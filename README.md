# BrainHaven 

**Ignite Your Tech Journey**  
Your ultimate platform for learning resources, codes, roadmaps, hackathon updates, team collaboration through Hackmate, and more.

---

## 🌟 Features

### **1. Resources**
- Access a vast collection of learning resources.
- Categorize and discover materials to accelerate your tech journey.

### **2. Code**
- Explore a comprehensive library of code snippets and detailed explanations.
- Includes solutions for various programming challenges and projects.
- Enhance your development skills with code examples.

### **3. Roadmaps**
- Discover detailed roadmaps for programming languages and career paths.
- Visual guides to help you navigate technologies, tools, and professional growth.
- Roadmaps include intuitive images for easy understanding.

### **4. Hackathons**
- Stay updated with the latest running hackathons.
- Direct links for easy participation.
- Never miss an opportunity to showcase your skills.

### **5. Hackmate**
- **Team Collaboration:** Create or join hackathon teams.
- **Request Management:** Admins can accept/reject requests, kick members, and manage teams.
- **Chat:** Personal chat for team members to coordinate effectively.
- **Member Actions:** Members can leave teams anytime.

### **6. User Profiles**
- Manage personal profiles and social links using dashboard.
- View and enhance your profile or explore other user's profiles.

### **7. Community Chat**
- Global chat accessible via the bottom-right button.
- Talk with all BrainHaven users at any time, fostering a vibrant community.

### **8. Admin Panel**
- Exclusive access for admins:
  - Manage all users.
  - Upload, edit, and delete resources.
  - Upload, edit, and delete code snippets.
  - Upload, edit, and delete hackathon information.
  - View and delete Hackmate teams.

---

## 🛠️ Tech Stack

### **Frontend**
- **React.js** with **Tailwind CSS**
- Responsive design for a seamless user experience.

### **Backend**
- **Node.js** with **Express.js**
- **Socket.io** for real-time chat functionality.

### **Database**
- **MongoDB** with **Mongoose**

### **Hosting & Storage**
- **Cloudinary** for image uploads and storage.

---

## 📦 Dependencies
- **bcryptjs**: Secure password hashing.
- **cloudinary**: Image and file management.
- **cors**: Cross-Origin Resource Sharing.
- **dotenv**: Environment variable management.
- **express**: Backend framework.
- **handlebars**: Template rendering.
- **helmet**: Enhanced security.
- **jsonwebtoken**: Authentication via JWT.
- **mongoose**: MongoDB object modeling.
- **multer & multer-storage-cloudinary**: File uploads and cloud storage.
- **nodemailer**: Email notifications.
- **nodemon**: Development monitoring.
- **socket.io**: Real-time communication.
- **uuid**: Unique ID generation.
